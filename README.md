Ali Rent A Car

A full-stack Car Rental Management System built using PHP, MySQL, Bootstrap 5, and Stripe.

 Live Demo: https://alimansoor.infinityfreeapp.com/car_rental/register.php

> Overview

Ali Rent A Car allows users to:
Browse and filter cars
Book rentals with or without a driver
Make payments (Stripe, Cash, EasyPaisa, JazzCash)
View invoices and leave reviews

Admins can manage cars, bookings, returns, and view reports.

 Tech Stack
Backend: PHP (Core)
Database: MySQL
Frontend: Bootstrap 5, JavaScript
Payments: Stripe
Email: PHPMailer
✨ Features
🔐 Authentication system
🚗 Car browsing with filters & search
📅 Booking system with price calculation
💳 Multiple payment methods
🧾 Invoice generation
⭐ Reviews & ratings
📊 Admin dashboard & reports
🌙 Dark mode
⚙️ Setup
Clone the repo
Move to htdocs (XAMPP)
Create database: car_rental_db
Import SQL file

Run:

http://localhost/car_rental/login.php
